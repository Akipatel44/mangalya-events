# Mangalya Event Management Website

A premium, modern event management website built with **Next.js 14**, **React**, **Tailwind CSS**, and **Framer Motion**.

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deploy on Vercel

### Option 1: One-Click Deploy
1. Push this project to a GitHub/GitLab/Bitbucket repository
2. Go to [vercel.com](https://vercel.com) and sign in
3. Click **"New Project"**
4. Import your repository
5. Vercel will auto-detect Next.js — click **Deploy**

### Option 2: Vercel CLI
```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel
```

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Deployment:** Vercel

## Pages

| Page | Route | Description |
|------|-------|-------------|
| Home | `/` | Hero, services, gallery, events, testimonials, CTA |
| About | `/about` | Company story, mission/vision, why choose us, team |
| Services | `/services` | Detailed service descriptions with features |
| Gallery | `/gallery` | Filterable masonry gallery with lightbox |
| Events | `/events` | Featured and past events |
| Testimonials | `/testimonials` | Client reviews with ratings |
| Contact | `/contact` | Contact form, info, and Google Maps embed |
